# 🎢 Thomas XL-200's Coaster Credit Tracker

A detailed roller coaster credit tracking app with stats, charts, and PNG export functionality.

## Features

- **Hybrid Serverless JSON Database**: Clean separation of UI logic, master coaster data, and user profile data.
- Track roller coaster credits across parks and countries.
- Rate, rank, and comment on every coaster.
- Add fully custom coasters and manufacturers dynamically.
- Detailed stats and visualizations (Comparison & Over Time charts).
- Customizable legend colors on Over Time Filters chart.
- Pixel-perfect PNG chart export (identical on mobile & desktop).
- Advanced mobile drag-and-drop mechanics with placeholder swap logic.
- Dark/light theme support.
- Multi-sort spreadsheet table view (up to 3 columns).
- Smart Export/import data for syncing across machines (generates timestamped backups).
- PWA-ready with offline support and mobile-optimized UI.
- All dynamic data is stored safely locally in the browser.
